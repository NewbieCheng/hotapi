# 激活码管理控制台 — Product Context

## Register

**product** — 内部运营后台，设计服务于任务效率而非品牌展示。

## Audience

- 内部运营 / 管理员
- 在桌面或手机上快速筛选、生成、复制、编辑激活码与权限
- 熟悉 SaaS 后台工具，期望 Linear / Stripe 级别的组件一致性

## Product

四插件激活码统一编排控制台（知发 / 知源 / 知聊 / 知销），对接 Vercel 部署的 `/api/activation` 管理端 API。

## Core tasks

1. 按插件切换 → 筛选密钥列表（状态、过期、搜索）
2. 批量生成激活码 → 复制 / 导出
3. 单条编辑权限、续期、删除
4. 系统选项：API 基址、列表偏好

## Surfaces

| 路由 | 说明 |
|------|------|
| `/admin/` 登录 | 管理密钥 + API 基址 |
| `/admin/` 控制台 | 插件 Tab + 列表 / 生成 / 设置 |

## Voice

- 中文、句子式、动作导向
- 控件名 = 用户能识别的结果（「复制 3 个」而非「批量操作」）
- 错误说明原因与下一步

## Anti-references

- 泛 SaaS 紫渐变 hero
- 卡片套卡片的信息架构
- 隐藏「自动」视图逻辑的黑盒切换
